## Local Development

### Prerequisites
- Node.js 16+ and npm

### Setup

1. Install dependencies:
```bash
npm install
cd frontend && npm install && cd ../backend && npm install
```

2. Start the backend:
```bash
cd backend
npm start
```

3. Start the frontend (in a new terminal):
```bash
cd frontend
npm run dev
```

4. Access the app:
- Frontend: http://localhost:5173
- Backend API: http://localhost:5000

### Seed Database
```bash
cd backend
npm run seed
```
